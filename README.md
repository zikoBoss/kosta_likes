# zakaria-outfit

موقع ويب لتوليد صورة الزي (Outfit) للاعب Free Fire باستخدام UID، بتصميم أحمر/أسود وخلفية رقمية.

## المميزات
- واجهة مستخدم عربية جذابة بألوان الأحمر والأسود.
- خلفية رقمية متحركة (نقاط).
- إدخال UID والمفتاح وعرض الصورة الناتجة مباشرة.
- جاهز للنشر على Render.

## النشر على Render
1. ادفع المشروع إلى GitHub.
2. في Render، اختر "New Web Service" واربط المستودع.
3. اختر البيئة "Python".
4. تأكد من أن أمر البدء هو: `gunicorn app:app`
5. أضف المتغير البيئي `MAIN_KEY` إذا أردت تغيير المفتاح الرئيسي.
6. اضغط "Create Web Service".

## المتغيرات البيئية
- `MAIN_KEY`: المفتاح السري للوصول إلى API (الافتراضي: CTX-TEAM).
- `PORT`: المنفذ الذي سيعمل عليه التطبيق (يُدار تلقائياً في Render).
